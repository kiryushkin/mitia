Каналы

Интеграции

Раздел /admin/integrations подключает сайт и внешние каналы. Настройки сохраняются для активного ассистента. После правок нажмите Сохранить.

Режимы. В большинстве карточек есть переключатели Ассистент и Оператор. Они взаимоисключающие: либо отвечает ИИ, либо работает операторский/автоответный режим.

Сайт (виджет)

  1. Откройте карточку «Сайт».
  2. Укажите домен, например https://example.com.
  3. Скопируйте готовый Script.
  4. Вставьте код на сайт перед </body>. Для Tilda: Настройки сайта → Ещё → HTML-код для body.
  5. Включите переключатель карточки. Система проверит, что скрипт реально установлен на сайте.
  6. Выберите режим: Ассистент или Оператор + текст автоответа.
  7. Нажмите Сохранить.

В коде установки используются client_id и при необходимости assistant_id. Виджет работает только на доменах из allowlist.

Если виджет не включается. Проверьте, что script опубликован на сайте, в нём верный client_id, домен указан без ошибок, и страница отдаётся по HTTPS.

Telegram

  1. В Telegram откройте @BotFather.
  2. Отправьте /newbot, задайте имя и username (должен заканчиваться на bot).
  3. Скопируйте API token.
  4. В MITIA откройте карточку Telegram, вставьте токен, включите карточку.
  5. Выберите режим Ассистент или Оператор, при необходимости задайте текст автоответа.
  6. Нажмите Сохранить.

После сохранения платформа регистрирует webhook на ваш бот. Сообщения из Telegram появятся в разделе Диалоги.

Max

  1. Получите токен бота Max.
  2. Вставьте его в карточку Max.
  3. Включите карточку — система проверит токен.
  4. Выберите режим и сохраните.

ВКонтакте

  1. Нужны права администратора сообщества и включённые сообщения сообщества.
  2. Получите ключ доступа с правами на сообщения.
  3. В карточке Вконтакте заполните: ключ доступа, ID сообщества, код подтверждения, секретный ключ.
  4. В настройках Callback API сообщества укажите URL вебхука MITIA: https://mitia.pro/api/chat/vk/webhook или .../webhook/{client_id}.
  5. Включите карточку, выберите режим и сохраните.

Email

  1. В карточке Email укажите почту и пароль (часто нужен пароль приложения).
  2. При необходимости заполните IMAP и SMTP. Для популярных провайдеров серверы подставляются автоматически.
  3. Включите карточку — система проверит авторизацию.
  4. Выберите режим Ассистент/Оператор.
Gmail / Yandex / Mail.ru. Обычно нужен пароль приложения и включённый доступ IMAP/SMTP во внешних приложениях.

Авито

  1. Создайте приложение Авито и получите Client ID / Client Secret.
  2. Вставьте их в карточку Авито.
  3. Включите карточку и сохраните.
  4. При необходимости включите синхронизацию сообщений.

Сообщения Авито обрабатываются через сервис платформы и попадают в общий inbox диалогов.

HeadHunter

  1. Откройте карточку HeadHunter.
  2. Нажмите Подключить через hh.ru.
  3. Авторизуйтесь в окне OAuth.
  4. Дождитесь статуса подключения и сохраните настройки.

Отключить интеграцию можно переключателем/действием disconnect в карточке.

Уведомления операторам

  1. Сначала подключите Telegram-бота в карточке Telegram — уведомления отправляются через его токен.
  2. Откройте карточку Уведомления.
  3. Включите нужные события: новые лиды, контакты, сообщения.
  4. Добавьте получателей: метка + Telegram user ID. ID можно узнать у @userinfobot.
  5. Включите карточку и сохраните.
