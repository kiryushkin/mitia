Панель

Панель управления

После входа открывается админ-панель MITIA: слева навигация, справа — содержимое выбранного раздела.

Как устроена панель

Панель работает как SPA. Основные адреса:

  • /admin — Профиль
  • /admin/assistant — Интеллект
  • /admin/settings — Внешний вид
  • /admin/integrations — Интеграции
  • /admin/dialogs — Диалоги
  • /admin/analytics — Аналитика
  • /admin/storage — Хранилище
  • /admin/tariffs — Тарифы
  • /admin/transactions — История пополнений
  • /admin/news — Новости

Навигация слева

  • Профиль
  • Интеллект
  • Внешний вид
  • Интеграции
  • Диалоги
  • Аналитика
  • Сохранить
  • Выйти

Внизу боковой панели — Сохранить и Выйти. В отдельных разделах появляются контекстные кнопки: назад, фильтры, удаление файлов, создание ассистента.

Как сохранять изменения

  1. Внесите правки в карточках раздела.
  2. Нажмите иконку дискеты «Сохранить» в левом меню.
  3. Дождитесь подтверждения успешного сохранения.
Важно. Без сохранения правки в Интеллекте, Внешнем виде и Интеграциях могут не примениться.

Рекомендуемый порядок запуска

  1. Интеллект — имя, роль, тон, база знаний, сайт, реквизиты.
  2. Внешний вид — виджет и окно чата под ваш бренд.
  3. Интеграции — сайт и/или мессенджеры.
  4. Диалоги — проверка реальных обращений.
  5. Профиль / Тарифы — лимиты, баланс, автопродление.
