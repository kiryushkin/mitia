Подключение Telegram

Интеграция с Telegram позволит получать сообщения от ваших клиентов, которые пишут в Telegram-бота, в приложение Mitia. Чтобы подключить Telegram, сначала нужно создать бота и сгенерировать ключ API.

Шаг 1: Создание бота

  1. Найдите в Telegram пользователя @BotFather и начните с ним чат.
  2. Отправьте пользователю сообщение /newbot для вызова команды создания нового бота.
  3. Придумайте и введите имя бота, которое будет отображаться для посетителей во время чата с ним.
  4. Введите username бота (обязательно должен заканчиваться на «bot», например: @assistant_mitia_bot).
  5. Скопируйте предоставленный ключ API (токен).

Шаг 2: Настройка интеграции

  1. Перейдите в раздел Интеграции в панели Mitia.
  2. Вставьте ключ API в поле «Токен бота» в карточке Telegram.
  3. Включите переключатель в заголовке карточки.
  4. Нажмите кнопку «Сохранить» (иконка дискеты в левом меню).

Параметры карточки Telegram

  • Автоматический ответ: отправляет мгновенное сообщение клиенту сразу после его обращения. Текст настраивается в поле под переключателем.
  • Включить ассистента: активирует ответы ИИ на основе вашей базы знаний. Если выключено — бот работает в режиме ручных ответов оператора.
  • Оператор: позволяет добавить ID пользователей для получения уведомлений о новых лидах в личные сообщения Telegram. Узнать свой ID можно у бота @userinfobot.

Интеграция с ВКонтакте

Для подключения сообщества ВК необходимо:

  • Быть администратором сообщества.
  • Разрешить сообщения в настройках группы.
  • Получить ключ доступа (Access Token) с правами на сообщения.
