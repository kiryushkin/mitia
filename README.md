# Mitia AI SaaS (Public Core)

Mitia — мультиарендная SaaS-платформа ассистента на FastAPI.
Проект включает чат-сценарии, админ-панель, авторизацию, биллинг и интеграции с внешними каналами.

## Что в этом репозитории

Репозиторий содержит публичное (неконфиденциальное) ядро проекта для хакатонной отправки:

- backend API и роутеры
- core-сервисы и бизнес-логика
- шаблоны и статические ассеты
- исходники и сборка виджета
- тесты и конфигурация деплоя

В git осознанно не включаются:

- реальные секреты и учетные данные
- локальные env-файлы (`.env*`, кроме `.env.example`)
- загрузки, логи, кэши, векторные индексы
- приватные ключи и сертификаты

## Технологии

- Python 3.11+
- FastAPI + Uvicorn/Gunicorn
- PostgreSQL (asyncpg + SQLAlchemy)
- Redis
- Jinja2 + frontend static assets
- Alembic
- Docker / Docker Compose

## Структура

- `api/chat/main_async.py` — основная точка входа FastAPI
- `api/chat/routers/` — API-роутеры и интеграции
- `api/chat/services/` — доменные сервисы
- `api/chat/core/` — конфигурация и общая core-логика
- `api/chat/templates/` — серверные HTML-шаблоны
- `api/chat/static/` — статические frontend-ресурсы
- `api/chat/tests/` — тесты

## Переменные окружения

Используйте шаблон:

```bash
cp .env.example .env
```

Заполняйте значения только для локального/серверного окружения.
Не коммитьте `.env` с реальными токенами.

## Локальный запуск (без Docker)

```bash
python3 -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python -m api.chat.main_async
```

Локальный URL по умолчанию:

- `http://127.0.0.1:5007`

## Запуск через Docker Compose

```bash
docker compose up --build
```

Сервисы описаны в `docker-compose.yml`.

## Безопасность перед публикацией

1. Перевыпустите токены, если они когда-либо использовались в локальных файлах.
2. Проверьте, что в коммитах нет реальных секретов.
3. В публичный репозиторий отправляйте только неконфиденциальное ядро проекта.
4. Не публикуйте приватные ключи, сертификаты и доступы к прод-сервисам.
